前言

以前对这个问题非常头疼，实际上是被错误的思路误导，总想着将腾讯客户端下载下来的专有格式转换成 .mp4 格式。总是到处找工具，很麻烦而且被广告困扰。

今天发现一种硬核的方式，能直接从网页中下载非加密格式的原始视频，分享给大家。

方法指北

首先进入视频网页选择好清晰度后，进入开发者模式。可以右单击检查键或者按下键盘上的 F12 。

然后在开发者模式中选择 Network ，将 size 排序模式设为降序。

然后 F5 刷新页面，找出尺寸最大的那一条路径，右单击复制链接。把链接粘贴进迅雷等软件，就可以获得视频。

是不是非常的简单鸭~~~

一点启发

按照此方法，是不是所有的类似视频网站，都可以按此方法下载呢？

留给读者探索。
